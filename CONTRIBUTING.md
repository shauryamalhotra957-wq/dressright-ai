# Contributing

Thanks for improving DressRight AI.

## Development setup

1. Fork or clone the repository.
2. Create a focused branch from `main`.
3. Install dependencies with `npm install`.
4. Start the service with `npm start`.

## Quality checks

Run `npm test` before opening a pull request. Add tests under `tests/` for behavior changes and keep fixtures free of personal or sensitive data.

## Pull requests

Describe the user impact, implementation, and validation. Include screenshots for interface changes and never commit credentials, private datasets, or generated dependency folders.
